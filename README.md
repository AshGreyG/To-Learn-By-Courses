# 📚 To-Learn-By-Courses

A personal learning repository documenting my journey through various computer
science and software engineering courses.

## 📖 Overview

This repository contains implementations, notes, and exercises completed while
following online courses and academic curricula. Each subdirectory represents an
independent learning path with self-contained code examples.

## 🚀 Running the Code

Each subdirectory is self-contained and can be run independently. See individual
directories for specific instructions.

## 🎯 Purpose

This repository serves as both a learning journal and a reference for future
projects. The code is written to be educational and may prioritize clarity over
optimization.
